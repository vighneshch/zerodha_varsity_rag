# 📚 RAG Pipeline with LangChain + Qdrant + HuggingFace + Llama3

A production-style **Retrieval-Augmented Generation (RAG)** system built using:
- LangChain
- Qdrant Vector Database
- HuggingFace Embeddings
- Cerebras LLM (Llama 3)
- PDF-based document ingestion

---

## 🚀 Features

- 📄 PDF document ingestion
- ✂️ Text chunking with overlap
- 🔢 Embedding generation using HuggingFace models
- 🧠 Vector storage in Qdrant
- 🔍 Semantic search (retrieval)
- 🤖 LLM-based answer generation
- 🌐 FastAPI-ready architecture
